# Board SPA
## 기능 
* 읽기
* 쓰기
* 수정하기
* 삭제하기
### 게시물 목록
* 게시물 검색
  * 검색 단어 입력창
  * 검색 버튼
* 게시물수 및 페이지수 표시
* 게시물 목록 타입 선택
  * 텍스트 형식
  * 썸네일 형식
* 게시물 목록
  * 게시물 번호
  * 제목
  * 작성자
  * 작성일
  * 첨부파일 다운
  * 조회수
* 게시물 목록 기능 버튼
  * 글쓰기
  * 페이지 이동
### 게시물 상세보기
* 게시물 제목
* 작성일
* 작성자
* 내용
* 첨부파일 목록
* 게시물 이동
  * 이전글
  * 다음글
* 게시물 상세보기 기능 버튼
  * 수정
  * 삭제
  * 목록
### 게시물 작성하기
* 제목
* 작성일 - 읽기전용
* 공지기간
  * 공지시작일
  * 공지종료일
* 작성자 - 읽기전용
* 내용
* 첨부파일
  * 파일선택
  * 첨부파일 폼 추가
  * 첨부파일 폼 삭제
* 게시물 작성하기 기능 버튼
  * 저장
  * 취소
  * 목록
### 게시물 수정하기
* 제목 - 수정가능
* 작성일 - 읽기전용
* 공지기간 - 수정가능
* 작성자 - 읽기전용
* 내용 - 수정가능
* 첨부파일
  * 기존 첨부파일 목록 표시
  * 기존 첨부파일 삭제
  * 새 첨부파일 선택
  * 첨부파일 폼 추가
  * 첨부파일 폼 삭제
* 게시물 수정하기 기능 버튼
  * 저장
  * 취소
  * 목록
### 게시물 삭제하기
* 삭제 버튼 클릭시 삭제 확인 alert
