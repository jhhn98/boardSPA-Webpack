# boardSPA: React + Webpack

## 프로젝트 구조

```bash
    ├── node_modules
    ├── public
    │   └── index.html
    ├── src
    │   ├── assets
    │   ├── components
    │   ├── data // 임시 데이터
    │   ├── styles
    │   ├── App.tsx
    │   ├── main.tsx
    ├── package.json
    └── webpack.config.cjs
```

## 설치 및 실행 (Installation & Run)

### 1) 패키지 설치

```bash
   npm install
```

### 2) 개발 서버 실행

```bash
   npm run dev
```

### 3) 프로덕션 빌드

```bash
   npm run build
```

### 4) 로컬 서버 접속

```bash
   http://localhost:5173
```
