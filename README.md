# boardSPA
## 목적
* 기존 MPA 구조의 게시판을 React 기반 SPA로 재구성하여,<br/>
  페이지 전환 없는 사용자 경험과 컴포넌트 중심 아키텍처를 이해한다.
* 게시글 조회, 작성, 수정, 삭제 등 게시판의 핵심 기능을<br/>
  프론트엔드–백엔드 분리 구조로 구현하며 데이터 흐름을 학습한다.
* UI 컴포넌트 구조화, 상태 관리, 라우팅, API 연동 등<br/>
  실무에 가까운 프론트엔드 개발 경험을 축적하는 것을 목표로 한다.
